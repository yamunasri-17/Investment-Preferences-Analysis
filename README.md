# Investment Behaviour and Preferences

This project focuses on analyzing investment behaviour and preferences using real-world data. 
The objective of this project is to understand how factors such as age, income, and risk appetite 
influence investment decisions.

## Tools Used
- Microsoft Excel
- Power BI

## Key Analysis Performed
- Data cleaning and preparation
- Analysis of investor age groups
- Risk vs return preference analysis
- Investment type comparison

## Dashboard Overview
The dashboard presents insights using bar charts, pie charts, donut charts, and KPIs 
to clearly visualize investment patterns and preferences.

## Conclusion
The analysis shows that most investors prefer low to medium risk investments, and 
long-term investment options are more popular among different age groups.
